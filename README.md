# Terminal 1: Watch SCSS

### Pre-PR Checklist

1. Compile SCSS: `npm run sass`
2. Check for JS errors in browser console
3. Test responsiveness on multiple screen sizes
4. Validate HTML (W3C validator)
5. Test all interactive features
